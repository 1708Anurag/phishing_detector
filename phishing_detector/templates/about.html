About this project

PhishGuard is a major academic project that detects phishing emails by combining a rule-based heuristic engine with a machine-learning text classifier. It is built with Python, Flask, SQLAlchemy (SQLite) and scikit-learn.

1. Rule-based heuristics

Scans sender address, subject and body for urgency language, credential requests, suspicious links (raw IPs, risky TLDs, lookalike brand domains) and mismatched display text.

2. ML text classifier

A TF-IDF vectorizer + Logistic Regression model trained on labeled phishing/legitimate samples estimates a phishing probability from the email's wording and style.

3. Combined risk score

The final 0–100 score blends 55% heuristic score with 45% ML probability, then maps to Safe / Suspicious / Phishing.

4. Persistence & auth

Flask-Login handles authentication; every scan is stored per-user in SQLite via SQLAlchemy for dashboards and history.
